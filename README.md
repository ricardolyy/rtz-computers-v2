

# 🖥️ RTZ COMPUTERS

Projeto acadêmico de desenvolvimento web front-end, criado para a disciplina de Construcao De Software Para Web da faculdade.

---

## 📌 Sobre o Projeto

A **RTZ COMPUTERS** é uma loja virtual fictícia especializada em periféricos gamer, desenvolvida com o objetivo de aplicar na prática os conhecimentos adquiridos em sala de aula, incluindo estruturação de páginas HTML, estilização com CSS, responsividade com Bootstrap, e aplicação mais interativa utilizando JavaScript e manipulação do DOM.

O site apresenta uma identidade visual voltada ao público gamer, com tema escuro e opcao de modo claro, efeitos neon em ciano e animações modernas.

---

## 🎯 Objetivos do Projeto

- Desenvolver uma página web completa do zero
- Aplicar conceitos de HTML5 semântico
- Estilizar com CSS3 utilizando animações, gradientes e efeitos visuais
- Garantir responsividade com o framework **Bootstrap 5**
- Aplicar interacao com JavaScript 
- Publicar o projeto em produção via **GitHub Pages**

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Finalidade |
|---|---|
| HTML5 | Estrutura da página |
| CSS3 | Estilização e animações |
| Bootstrap 5 | Responsividade e grid system |
| JavaScript | Interacao e funcionalidades dinamicas |
| GitHub Pages | Hospedagem e publicação |

---

## 📱 Responsividade

O projeto foi adaptado para diferentes tamanhos de tela utilizando o sistema de grid do Bootstrap 5:

- **Desktop** → layout completo com 3 colunas de produtos e 4 categorias
- **Tablet** → 2 colunas de produtos e 2 categorias
- **Mobile** → layout empilhado, menu hambúrguer e pesquisa adaptada
